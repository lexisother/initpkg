# `initpkg` [![npm version](https://badge.fury.io/js/initpkg.svg)](https://badge.fury.io/js/initpkg)

Personal CLI to init new projects

## Installation & Usage

```bash
npm init @keanucode/project
# or
npx @keanucode/initpkg
# or
npm install -g @keanucode/initpkg
initpkg
```
